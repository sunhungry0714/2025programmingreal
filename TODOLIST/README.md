# TODO List

이 프로젝트는 Streamlit을 사용하여 제작된 간단한 할 일 목록 웹 애플리케이션입니다.

## 주요 기능
- 할 일 목록 보기
- 새로운 할 일 추가
- 기존 할 일 삭제

## 실행 방법
1. Python과 Streamlit이 설치되어 있는지 확인합니다.
2. 아래 명령어를 실행하여 애플리케이션을 시작합니다:

```bash
streamlit run app.py
```

3. 웹 브라우저에서 애플리케이션을 확인할 수 있습니다.

## 요구 사항
- Python 3.7 이상
- Streamlit 라이브러리
